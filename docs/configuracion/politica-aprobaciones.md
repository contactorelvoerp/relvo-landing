# Cómo configurar la política de aprobaciones

La política global de aprobación define el comportamiento base de aprobaciones para toda la organización. Esta vista es el punto correcto para fijar reglas que luego se heredan en los contratos.

## Antes de comenzar

Usa esta vista si quieres establecer una política estándar y evitar configurar cada contrato desde cero. Solo conviene bajar al nivel de contrato cuando realmente necesitas una excepción.

## Qué puedes configurar

En la política global de aprobaciones puedes definir si la aprobación interna está activa por defecto, si la OC es obligatoria por defecto, quién puede aprobar y a quiénes se notifica.

La configuración que dejes aquí se hereda en contratos, salvo que un contrato tenga un override específico.
