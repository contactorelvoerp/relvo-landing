# Configurar la organización

La sección de configuración es el hub administrativo de la organización. Aquí revisas reglas globales que después se heredan en otras partes del producto, especialmente en aprobaciones y facturación.

## Cuándo usar esta sección

Usa `Configuración` al inicio de una implementación o cuando necesites ajustar reglas base para toda la organización. Estas decisiones evitan depender de cambios manuales contrato por contrato.

## Qué muestra la vista principal

La vista principal de configuración organiza tarjetas para empresa, usuarios, aprobaciones, plan de cuentas, preferencias de facturación e integraciones.

> **Nota:** No todas las tarjetas visibles están operativas con backend completo. Puedes verlas en la interfaz, pero algunas siguen en borrador o dependen de desarrollo adicional.

## En esta sección encuentras

1. [Política de aprobaciones](./politica-aprobaciones.md): cómo definir la configuración base de aprobación para toda la organización.
2. [Preferencias de facturación](./preferencias-facturacion.md): cómo ajustar reglas globales de emisión antes de operar.
