# Cómo funciona Relvo

Relvo transforma acuerdos comerciales en reglas operativas de cobro.

En vez de calcular manualmente cuánto cobrarle a cada cliente, Relvo usa la configuración del contrato y el consumo real del periodo para generar montos a facturar.

## Flujo general

El flujo base de Relvo es:

1. Contrato comercial.
2. Configuración en Relvo.
3. Uso o consumo del periodo.
4. Cálculo de cargos.
5. Prefactura.
6. Aprobación.
7. Factura.
8. Conciliación.

Para ver el recorrido completo, continúa en [Flujo quote-to-cash](./flujo-quote-to-cash.md).

Para pasar a la configuración inicial, revisa [Primeros pasos](./primeros-pasos.md).
