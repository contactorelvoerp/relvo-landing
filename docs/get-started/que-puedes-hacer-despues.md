# Qué puedes hacer después

Después de entender qué es Relvo y cómo se organiza el flujo quote-to-cash, el siguiente paso es configurar una operación inicial en la plataforma.

## Si estás partiendo desde cero

Sigue [Primeros pasos](./primeros-pasos.md). Esa guía te lleva por el orden recomendado:

1. Crear cliente.
2. Crear cuenta de facturación.
3. Crear items y métricas de cobro.
4. Crear un plan si vas a reutilizar pricing.
5. Crear contrato.
6. Registrar o importar uso.
7. Revisar próximos cobros y facturación.

## Si ya tienes clientes y contratos

Revisa primero qué falta para que Relvo pueda calcular y preparar cobros:

- Cuenta de facturación completa.
- Contrato activo.
- Cargos configurados.
- Métricas para cargos por uso.
- Uso cargado para el periodo.
- OC, HES o aprobaciones si el contrato las requiere.
- FX definido si precio y facturación usan monedas distintas.

## Si estás validando el proceso con un equipo

Usa este recorrido para una capacitación inicial:

1. [Qué es Relvo](./overview.md).
2. [Flujo quote-to-cash](./flujo-quote-to-cash.md).
3. [Conceptos principales](./conceptos-principales.md).
4. [Primeros pasos](./primeros-pasos.md).
5. [Cobros y facturación](./revisar-cobros-facturacion.md).

> **Base operativa:** cliente, cuenta de facturación, contrato, cargos, uso y próximos cobros son los elementos mínimos para operar revenue en Relvo.
