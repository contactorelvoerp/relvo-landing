# Ver un cliente

El detalle de cliente es la vista 360 de la cuenta. Te permite revisar su estado operativo y navegar entre contratos, uso y facturación sin perder el contexto del mismo cliente.

## Cuándo usar esta vista

Usa esta vista cuando quieras entender si el cliente ya está listo para ser facturado y qué información le falta. También es la mejor forma de revisar la cuenta completa antes de intervenir en un contrato o una factura.

## Cómo está organizada la vista

La vista de cliente se organiza en tabs:

1. `Resumen`: muestra el estado operativo, la readiness para facturar y estadísticas clave.
2. `Cuentas de facturación`: reúne perfiles y datos legales para emitir.
3. `Contratos`: muestra los contratos vinculados al cliente.
4. `Uso`: concentra el consumo reciente del cliente.
5. `Facturación`: reúne facturas, vencimientos y saldo abierto.

Empieza por `Resumen` si necesitas una lectura rápida del estado de la cuenta. Después entra al tab específico según lo que quieras resolver, por ejemplo contratos si vas a revisar condiciones comerciales o facturación si quieres ver documentos y vencimientos.
