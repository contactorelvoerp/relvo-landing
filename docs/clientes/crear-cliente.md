# Cómo crear un cliente

La vista `Nuevo cliente` se usa para dar de alta una cuenta comercial en Relvo. Es el punto de partida para después asociar contratos, uso y facturación a esa cuenta.

## Antes de comenzar

Antes de crear el cliente, ten a mano sus datos principales, la dirección y el contexto comercial. Si ya conoces la configuración legal con la que vas a emitir, también puedes cargar una cuenta de facturación inicial.

## Completar el alta

1. Entra a la opción para crear un cliente nuevo desde `Clientes`.
2. Completa los datos del cliente, su dirección y el contexto comercial.
3. Si corresponde, agrega una cuenta de facturación inicial como parte del alta.
4. Guarda el registro para dejar disponible al cliente dentro de la bandeja principal.

Después de crear el cliente, normalmente el siguiente paso es revisar su detalle para validar si ya tiene todo lo necesario para operar y facturar.
