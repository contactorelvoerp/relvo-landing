# Trabajar con clientes

La sección de clientes es la bandeja principal para buscar cuentas comerciales, revisar su estado y entrar al contexto completo de cada una. Desde aquí puedes crear un cliente nuevo o entender si ya está listo para ser facturado.

## Cuándo usar esta sección

Usa `Clientes` cuando necesites partir desde la cuenta comercial y no desde un contrato específico. También es el mejor punto de entrada si quieres ver contratos, uso y facturación sin salir del contexto del cliente.

## En esta sección encuentras

1. [Crear un cliente](./crear-cliente.md): cómo dar de alta una nueva cuenta comercial y, si corresponde, agregar una cuenta de facturación inicial.
2. [Ver un cliente](./detalle-cliente.md): cómo usar la vista 360 del cliente y sus tabs principales.

## Qué puedes hacer en la bandeja de clientes

En la bandeja de clientes puedes buscar por nombre, RUT o estado, revisar cuántos contratos activos tiene cada cuenta, ver la próxima factura y abrir la vista de un cliente específico. También puedes iniciar la creación de un cliente nuevo desde esta misma sección.
