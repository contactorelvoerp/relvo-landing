# Ver una factura

El detalle de factura se usa para revisar un documento junto con su contexto operativo. Es la vista adecuada cuando necesitas entender no solo el monto, sino también el workflow, la relación con el cliente y el contrato.

## Cuándo usar esta vista

Usa la vista de factura cuando debas revisar líneas, montos, aprobaciones, eventos del timeline o información de OC. También es útil para confirmar qué originó la factura y en qué estado quedó el documento.

## Qué puedes ver

En esta vista encuentras el resumen principal del documento, sus líneas y montos, el workflow y las aprobaciones, el timeline y los eventos, las relaciones con cliente y contrato, y la información de OC cuando aplique.
