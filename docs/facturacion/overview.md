# Trabajar con facturación

La sección de facturación es la bandeja operativa para emitir y seguir facturas. Piensa esta vista como el inbox principal de facturación dentro de Relvo.

## Cuándo usar esta sección

Usa `Facturación` cuando necesites revisar el estado de un documento, procesar facturación por contrato o validar qué facturas siguen detenidas en el workflow.

## Qué puedes hacer en la bandeja de facturación

En la bandeja de facturación puedes filtrar por estado de factura, estado del workflow, cliente, origen o necesidad de OC. También puedes abrir una preview draft y procesar facturación por contrato.

> **Nota:** La creación manual de facturas aparece hoy en la interfaz como una capacidad futura. Puedes verla en la vista, pero no está descrita como un flujo operativo completo en la guía base.
