# Cómo agregar un evento de uso

La vista `Agregar evento de uso` se usa para registrar un evento puntual sin cargar un archivo. Es útil cuando necesitas corregir o cargar consumo de forma manual.

## Antes de comenzar

Antes de ingresar el evento, confirma qué cliente corresponde, qué métrica se va a usar y qué fecha y hora debe quedar registradas. También conviene tener claras las propiedades del evento si forman parte del contexto de medición.

## Registrar el evento

1. Entra a la opción para agregar un evento desde `Uso`.
2. Selecciona el cliente.
3. Define la métrica y el evento.
4. Completa la fecha y hora.
5. Agrega las propiedades del evento.
6. Guarda el registro para que quede disponible en la bandeja de uso.

Si necesitas cargar muchos eventos al mismo tiempo, usa en cambio la importación por CSV.
