# Cómo entender y revisar items

Los `Items` son los productos o servicios base del catálogo comercial. Esta vista te sirve para revisar esa base y entender cómo impacta en los planes que luego usarán los contratos.

## Cuándo usar esta vista

Usa el tab `Items` dentro de `Catalogo y Pricing` cuando necesites revisar la estructura comercial base antes de trabajar métricas o planes. Si ya estás evaluando un item concreto, entra a su detalle para revisar o ajustar su configuración.

## Revisar el detalle de un item

La vista de item se usa para revisar o ajustar un item del catálogo y entender su impacto en planes. Si vas a modificar pricing reutilizable, este detalle te ayuda a validar primero qué componente base estás tocando.
