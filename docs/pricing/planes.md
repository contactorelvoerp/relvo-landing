# Cómo crear y revisar planes

Los planes son estructuras reutilizables de pricing. Sirven para agrupar cargos y condiciones que luego heredarán los contratos nuevos.

## Cuándo usar esta sección

Usa `Planes` cuando necesites definir una estructura comercial reusable o revisar el snapshot vigente de un plan ya existente. Esto te ayuda a mantener consistencia entre contratos que comparten una misma lógica de cobro.

## Crear un plan

Al crear un plan, completas la identidad del plan, la moneda, el compromiso mínimo cuando aplica, los cargos recurrentes, los cargos únicos, los cargos por uso y las fases del plan.

## Revisar un plan existente

En la vista de plan puedes revisar el snapshot vigente del plan y la composición de cargos que heredarán los contratos nuevos. Si primero necesitas encontrar el plan correcto, usa la lista de planes, donde puedes buscar, abrir uno existente o crear uno nuevo.
