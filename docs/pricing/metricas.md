# Cómo crear una métrica de cobro

Las métricas de cobro definen cómo Relvo mide el consumo y lo transforma en una cantidad facturable. Esta configuración es clave cuando el cobro depende de eventos de uso.

## Antes de comenzar

Antes de crear una métrica, ten claro qué quieres medir, cómo se va a comportar esa medición y qué filtros o dimensiones necesitas para distinguir distintos casos operativos.

## Crear la métrica

1. Entra a la opción para crear una métrica desde `Pricing`.
2. Define la identidad comercial de la métrica.
3. Configura el comportamiento de medición.
4. Ajusta filtros y dimensiones.
5. Revisa la vista previa operativa antes de dejarla lista para usar.

Una vez creada, la métrica pasa a formar parte de la lógica comercial disponible para pricing y uso.
