# Trabajar con aprobaciones

La sección de aprobaciones es la bandeja de trabajo para aprobadores y operadores. Desde aquí puedes resolver pasos pendientes, registrar OC y revisar el historial de acciones ya cerradas.

## Cuándo usar esta sección

Usa `Aprobaciones` cuando una factura o un contrato no pueden avanzar automáticamente y necesitas intervenir en el workflow. También te sirve para documentar decisiones mediante notas y entrar al contrato relacionado.

## Cómo está organizada la vista

La vista de aprobaciones tiene tres tabs:

1. `Aprobaciones internas`: pasos pendientes de aprobación interna.
2. `OC`: solicitudes de orden de compra o aprobación externa.
3. `Historial`: pasos ya resueltos.

## Qué puedes hacer

Desde esta bandeja normalmente puedes aprobar o rechazar pasos, registrar una OC, dejar notas y entrar al contrato relacionado para revisar el contexto completo del acuerdo.
