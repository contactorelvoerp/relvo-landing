# Cómo crear un contrato

La vista `Nuevo contrato` se usa para crear el acuerdo comercial que después alimenta la facturación. Aquí defines las condiciones base del contrato antes de empezar a operarlo.

## Antes de comenzar

Antes de crear el contrato, conviene tener identificado el cliente asociado y las condiciones comerciales que vas a registrar. También necesitas claridad sobre vigencia, frecuencia de facturación y requisitos de aprobación cuando apliquen.

## Completar el contrato

1. Entra a la opción para crear un contrato nuevo desde `Contratos`.
2. Selecciona el cliente asociado.
3. Define fechas y vigencia.
4. Configura cargos y condiciones comerciales.
5. Indica la frecuencia de facturación.
6. Agrega aprobaciones y requisitos externos cuando corresponda.
7. Guarda el contrato para dejarlo disponible en la bandeja de contratos.

Una vez creado, el contrato pasa a ser el punto central para revisar facturas asociadas, workflow de aprobación y documentos del acuerdo.
