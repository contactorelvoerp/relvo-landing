# Cómo editar un contrato

La vista `Editar contrato` se usa para modificar contratos editables o en borrador. Sirve para ajustar el acuerdo antes de dejarlo listo para operar.

## Antes de comenzar

Usa esta vista cuando todavía estás afinando el contrato y necesitas corregir fechas, cargos, frecuencia de facturación o requisitos de aprobación. Si el contrato ya no es editable, conviene revisar primero su detalle para entender su estado actual.

## Qué revisar al editar

Al editar un contrato, vuelve a validar que el cliente asociado, la vigencia, las condiciones comerciales y las reglas de facturación reflejen el acuerdo que quieres operar. Si el contrato depende de aprobaciones o requisitos externos, revisa también que esa configuración siga siendo correcta antes de guardar los cambios.
